# 01. Geometry of a Line

## Core Idea

Machine Learning learns patterns, not tables.

So, every row in a dataset is represented as a point on a graph.

Dataset → Points → Pattern

---

## Cartesian Coordinate System

- X-axis → Input (Feature)
- Y-axis → Output (Target)
- Origin → (0,0)

Each point is represented as:

(x, y)

Order matters.

---

## One Row = One Point

Example:

Hours   Marks

2       40
4       60
6       80

↓

(2,40)
(4,60)
(6,80)

---

## Why Plot Data?

A graph reveals patterns that are difficult to see in a table.

It helps identify:
- Trends
- Outliers
- Relationships

---

## Straight Line

A straight line represents a **constant rate of change**.

Equal change in input
→ Equal change in output

---

## Connection to Linear Regression

Linear Regression converts:

Dataset
→ Points
→ Best Fit Line
→ Predictions

It learns the overall trend, not individual data points.

---

## Remember

One Row = One Point

Dataset → Geometry

Best Fit Line → Pattern → Prediction
