# 02. How Linear Regression Works

• Linear Regression learns patterns from historical data to predict numerical values.

Workflow:
Collect Data
→ Choose Features & Target
→ Train Model
→ Learn Pattern
→ Predict New Values
→ Evaluate Performance

Feature:
Input variable given to the model.
Example: House Area, Years of Experience.

Target:
Output variable the model predicts.
Example: House Price, Salary.

Training:
The process of teaching the model using historical data.

Prediction:
The estimated output for a new input.

Generalization:
The ability of a model to make good predictions on unseen data.

Key Idea:
The model learns the relationship between inputs and outputs instead of memorizing the dataset.

Remember:
• Features → Inputs
• Target → Output
• Training → Learning
• Prediction → Estimating
• Generalization → Performing well on unseen dataV
