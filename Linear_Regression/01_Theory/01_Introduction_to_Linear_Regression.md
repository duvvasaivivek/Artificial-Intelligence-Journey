# 01. Introduction to Linear Regression

Definition:
Linear Regression is a Supervised Machine Learning algorithm used to predict continuous numerical values by learning the relationship between input features and the target variable.

Why do we need it?
Used whenever we need to predict a number instead of a category.

Examples:
• House Price Prediction
• Salary Prediction
• Sales Forecasting
• Temperature Prediction
• Electricity Consumption

Core Idea:
Learn the overall pattern (trend) in the data and use it to predict unseen values.

Best Fit Line:
The straight line that represents the overall trend of the data as closely as possible.

Regression vs Classification

Regression
• Predicts numerical values
• Example: House Price, Salary, Marks

Classification
• Predicts categories
• Example: Spam/Not Spam, Dog/Cat

Why is it called "Linear"?
Because the relationship between inputs and output is approximated using a straight line.

Why is it called "Regression"?
Regression means predicting a continuous numerical value.

Important Terms

Feature (Input)
Information provided to the model.
Example: House Area

Target (Output)
Value the model predicts.
Example: House Price

Prediction
Estimated output produced by the model.

Trend
Overall relationship between input and output.

Key Points
• Supervised Learning algorithm
• Predicts continuous numerical values
• Learns patterns from historical data
• Uses the Best Fit Line for prediction
• Predictions are estimates, not exact values

Remember
Regression → Numbers
Classification → Categories
Feature → Input
Target → Output
