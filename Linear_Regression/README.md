# Machine Learning from Scratch

A structured, first-principles journey to mastering Machine Learning by understanding the mathematics, implementing algorithms from scratch, and then using industry-standard libraries.

Unlike traditional tutorials that jump directly into APIs, this repository is designed to help you build a deep understanding of how Machine Learning algorithms actually work.

---

## Philosophy

This repository follows a simple learning principle:

> **Understand the theory → Master the mathematics → Build it from scratch → Use industry libraries → Experiment → Apply it to real-world projects.**

Every topic is broken into small, modular learning units, making it easier to study, revise, and expand over time.

---

## Repository Goals

* Learn Machine Learning from first principles.
* Develop strong mathematical intuition.
* Implement algorithms without relying on ML libraries.
* Understand how industry tools work internally.
* Build clean, modular, and production-quality code.
* Strengthen problem-solving through experiments and practice.
* Create portfolio-worthy projects for every major algorithm.

---

## Learning Workflow

For every topic, follow this order:

1. Theory
2. Mathematics
3. Scratch Implementation
4. Visualizations
5. Library Implementation
6. Practice Problems
7. Experiments
8. Mini Project
9. Revision Notes

This ensures conceptual understanding before using high-level libraries.
